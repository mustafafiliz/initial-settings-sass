# Installation
Bu komut gerekli library'leri yükler.
> npm i

# Sass Intallation
Bu komut sass yükler.
> npm i -g sass

# Start
Bu komut localhost'u başlatır ve sass watch'ı aktif eder assets/sass dosyaları otomatik olarak assets/css klasörüne bundle edilir.
> npm run start

# Testing
> http://localhost:1234
